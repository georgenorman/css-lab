Basic Floats

Float Left No Frills

left is floated left and is 100px wide. Height is not set. It has no margin, border or padding. main is not floated. Height and width are not set. It has no margin, border or padding. Notice that the width of main has expanded to fill outermost, but the height only expands to contain its own content. The left div is essentially floated on top of main, in the upper-left corner and the non-block contents of main flows around it. The use of CSS for the content (e.g., .main::before {content: "Main Content..."}) does not produce selectable text. As a test, "This is a test" has been added to the main div to validate that normal text is selectable. float, height, width, margin, padding, border outermost_bf10, left_bf10, main_bf10

Float Left with Padding

left is the same as before, but this time it has some padding. The padding for left works as expected. main is the same as before, but now has padding. Notice that the padding has zero impact on the text that flows around left, but does affect the text anywhere it touches outermost. Also, the width of main has shrunk, to account for the padding. float, width, margin, padding, border outermost_bf20, left_bf20, main_bf20

Float Left with Padding and Margin

left is the same as before, but this time a right margin has been added to pad the main text. main is also the same as before, but this time its height has been set to 100%, to fill the outermost container. Notice that the padding, added by a previous experiment, has caused the background to overflow, now that the height is 100%. float, width, margin, padding, border outermost_bf30, left_bf30, main_bf30

Float Left with Overflow

left is the same as before. main is the same as before. outermost now has overflow:hidden. float, width, margin, padding, border outermost_bf40, left_bf40, main_bf40

Float Right with Overflow

Everything is the same as the previous experiment, except now left div has been replaced by right. Also, the left margin on the float has been changed to a right margin. float, width, margin, padding, border outermost_bf50, right_bf50, main_bf50

Compound Floats

Float Left and Right

This experiment builds on all of the previous experiments. It combines the float left and float right experiments into one. The main div must be last, otherwise, the floats will be under main. Overflow has now been changed to scroll. However, the background of main does not cover 100% of the area. float, width, margin, padding, border outermost_cf10, left_cf10, right_cf10, main_cf10
