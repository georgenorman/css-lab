Basic Box Model Sans Border

Removing the border from the outerMost div, causes the box model to collapse the top and bottom margins of the inner div.

Basic Box Model with Padding Sans Border

Adding some padding to the outerMost div can be used instead of a border. However, the width of the outer div will need to be reduced, to compensate for the missing border. Furthermore, the margin of the inner div will need to be reduced by the size of the outer padding.
